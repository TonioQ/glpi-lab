IT Help Desk Automation Lab (GLPI + Active Directory + Automation Scripts)

Goal: Build a realistic help desk environment using Ubuntu Server (GLPI + MariaDB) integrated with a 
Windows Active Directory domain. Add automation scripts for ticket creation, backups, and monitoring.

## Milestones
- VM base: Ubuntu Server installed, SSH enabled
- LAMP stack: Apache, PHP, MariaDB
- GLPI installed & reachable
- AD/LDAP authentication working
- Automation: ticket creation script
- Backups: DB + config export (cron)
- Documentation: screenshots + troubleshooting
